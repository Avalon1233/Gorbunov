# АСУ "Заработная плата сотрудников предприятия"

## Описание
Консольное приложение на C++ для управления заработной платой сотрудников предприятия с полной функциональностью редактирования, сортировки, фильтрации, поиска и агрегирующих запросов.

## Структура проекта

### Файлы
- **Employee.h / Employee.cpp** - Класс для описания сотрудника с 12 основными полями
- **EmployeeDatabase.h / EmployeeDatabase.cpp** - Класс для управления списком сотрудников (в памяти)
- **ConsoleApplication1.cpp** - Главное приложение с интерактивным меню

## Основные поля сотрудника
1. ФИО
2. Наименование цеха
3. Зарплата
4. Год рождения
5. Дата поступления на работу (ДД.ММ.ГГГГ)
6. Семейное положение
7. Пол (М/Ж)
8. Количество детей
9. Дата заболевания (ДД.ММ.ГГГГ)
10. Дата выздоровления (ДД.ММ.ГГГГ)
11. Оплата по бюллетеню (%)
12. Средний заработок

## Использование

При запуске приложения вы увидите главное меню с следующими опциями:

```
========== ГЛАВНОЕ МЕНЮ ==========
1. Управление сотрудниками
2. Сортировка
3. Фильтрация
4. Поиск
5. Агрегирующие запросы
6. Просмотр всех сотрудников
7. Подключение к БД Microsoft Access
0. Выход
```

Выберите нужный пункт и следуйте инструкциям на экране.

## Примеры использования

### Добавление сотрудника
1. Выберите "1. Управление сотрудниками"
2. Выберите "1. Добавить сотрудника"
3. Введите данные сотрудника по запросу

### Сортировка по зарплате
1. Выберите "2. Сортировка"
2. Выберите "2. По зарплате (убывание)"
3. Будет выведен отсортированный список

### Фильтрация по цеху
1. Выберите "3. Фильтрация"
2. Выберите "1. По цеху"
3. Введите название цеха
4. Будут выведены все сотрудники из этого цеха

### Получение статистики
1. Выберите "5. Агрегирующие запросы"
2. Выберите нужный запрос
3. Будет выведен результат
